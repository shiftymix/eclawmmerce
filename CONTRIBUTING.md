# Contributing to Council of eCLAWmmerce

Thanks for your interest in contributing! Here's how to get started.

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Install dependencies: `npm install`
4. Create a Supabase project and run migrations
5. Copy `.env.local` and fill in your values
6. Run `npm run dev` to start the dev server

## Pull Requests

- Create a feature branch from `main`
- Write clear commit messages
- Test your changes locally before submitting
- Keep PRs focused on a single change

## Code Style

- TypeScript strict mode
- Use `JetBrains Mono` for body text, `Press Start 2P` for headers only
- Follow the existing Tailwind patterns and color tokens
- Max border-radius of 4px (keep it boxy and retro)

## Reporting Issues

Open a GitHub issue with:
- A clear description of the bug or feature request
- Steps to reproduce (for bugs)
- Expected vs actual behavior
